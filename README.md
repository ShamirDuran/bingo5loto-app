# Bingo 5 Loto

Aplicación móvil desarrollada en Flutter que permite jugar al bingo.

Cada cartón se genera de forma aleatoria al igual que las balotas generadas.

Letras disponibles:

- T
- más
- X
- L

y el pleno.
